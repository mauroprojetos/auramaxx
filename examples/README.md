# Play games from your terminal

**Game engine from your terminal.** No install needed.

```bash
npm install -g auramaxx
auramaxx play aurasu
```

Run locally from this repo:

```bash
cd examples
npm run list
npm run aurasu
```

## Games

- [minecraft-voxel](../packages/aurascript/examples/minecraft-voxel) — Minecraft-style voxel sandbox with native 3D collision
- [aurasu](./aurasu) — A hit-circle rhythm game
- [brick-breaker](./brick-breaker) — Paddle-and-ball arcade game
- [subway-runner-3d](./subway-runner-3d) — 3D endless runner
- [platformer-3d](./platformer-3d) — 3D physics + collision vertical slice prototype
- [tier3-vertical-slice](./tier3-vertical-slice) — Tier-3 3D quality scene (shadows/reflections/water/billboards/joint-like chains)
- [walk-sprite](./walk-sprite) — Lane-runner with sprite sheets
- [3d-demo](./3d-demo) — 3D scene with camera controls and lighting
- [camera-rig](./camera-rig) — Camera follow/deadzone/effects demo
- [multiplayer-host](./multiplayer-host) — Host-authoritative multiplayer room (port `7000`)
- [multiplayer-client](./multiplayer-client) — Client input-forwarding demo (connects to `127.0.0.1:7000`)
